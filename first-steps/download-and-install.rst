Download and install
--------------------

.. toctree::

   install-all-in-one   

|PhaserEditor|_ is built with web technologies. You can run it on your desktop, your private network, or in the cloud.

We provide two different types of distributions: |core|_ and |allInOne|_. Each distribution is available for Windows, macOS, and Linux.

`Download Phaser Editor 2D <https://phasereditor2d.com/downloads>`_

In this chapter we cover how to install both distributions.
