SpineGameObject user properties
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

In |PhaserEditor|_ you can encapsulate game object behavior into `prefabs <prefabs.html>`_, `user components <user-components.html>`_, and `script nodes <script-node.html>`_. These mechanisms use the `user properties <prefab-user-properties.html>`_ to customize the behaviors.

For a better implementation of behaviors related to Spine_ game objects, we provide two user property types:

* `Spine Skin Name property type <prefab-user-properties-spine-skin-name.html>`_
* `Spine Animation Name property type <prefab-user-properties-spine-animation-name.html>`_
